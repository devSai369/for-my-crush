# 💌 From Venkat, With Love

A romantic interactive web page — built with pure HTML, CSS & JavaScript. No frameworks, no dependencies.

## ✨ Features

- Custom animated cursor with glow ring
- Falling rose petals & twinkling stars
- **5 beautiful sections:**
  - 💌 Intro with animated envelope
  - 🌸 Reasons (hover cards)
  - 📅 Milestone timeline
  - 📝 Handwritten love letter
  - 💛 Wish jar + interactive heart + Yes/No game
- Fireworks canvas animation
- Scroll progress bar
- Smooth section navigation dots
- Fully responsive (mobile-friendly)

## 🛠 How to deploy on GitHub Pages

1. **Create a new repository** on GitHub (e.g. `for-my-crush`)
2. Upload both files (`for-my-crush.html` and `README.md`)
3. **Rename** `for-my-crush.html` → `index.html`  
   *(GitHub Pages serves `index.html` automatically)*
4. Go to **Settings → Pages**
5. Under *Source*, select **Deploy from a branch**
6. Choose `main` branch → `/ (root)` → click **Save**
7. Wait ~1 minute, then visit:  
   `https://<your-username>.github.io/for-my-crush/`

That's it! Share the link 💝
